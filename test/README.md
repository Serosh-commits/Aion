# Aion Test Suite

This directory contains standard test cases for the LLVM Optimization Failure Debugger.

## Files
- `cases.cpp`: Source file containing various patterns that deliberately fail optimizations (loop vectorization, inlining, SROA, etc).
- `cases.ll`: LLVM IR generated from `cases.cpp`.
- `cases.opt.yaml`: Optimization remarks generated by Clang.

## Running Tests
To regenerate the test artifacts:
```bash
clang++ -O2 -g -fsave-optimization-record -emit-llvm -S cases.cpp -o cases.ll
```

To run the debugger against this suite:
```bash
../build/opt-debugger --before=cases.ll --after=cases.ll --remarks=cases.opt.yaml
```
